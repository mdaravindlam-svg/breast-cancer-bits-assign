# Breast Cancer Classification ML Project

## Problem Statement
Predict whether tumor is malignant or benign using ML models.

## Dataset
Breast Cancer Wisconsin dataset
- 569 samples
- 30 features
- Binary classification

## Models Used
- Logistic Regression
- Decision Tree
- KNN
- Naive Bayes
- Random Forest
- XGBoost
